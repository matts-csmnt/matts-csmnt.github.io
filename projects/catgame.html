[C#] The Cat Game

C# w/ Unity - Mobile App

The Cat Game aimed to be a fun mobile game (targetting Android devices) inspired by Lemmings, that also implemented assistive technology to help players who are usually unable to play mobile games because of a physical impairment.

The project was worked on by 5 final year BSc programming students, where the development focussed on rapid prototyping and implementations split between each member of the group.

Once the core gameplay was implemented, my job was to create post-effects shaders that would aid visually impaired players while playing the game. This included looking at contrast filters and colourblindnedd colour tinting.

I also worked closely with one other developer on the team to implement a UI scanning technique allowing players to play the game using a switch system (sometimes only one button if the players had trouble using more).
