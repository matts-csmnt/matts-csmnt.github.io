[C++/PS4] Hogs of War: Reheated - Graphics & Engine



Hogs of War was a heavily optimised PlayStation game that made use of Sony's proprietary libraries for everything from rendering to animation to loading files.
A port of this game would clearly need to be built from the ground up, not relying on existing game engine technologies.

I was brought on board this project to help create the PlayStation 4 Engine and Rendering technology.

The scope of this project has been huge, and because of this I've seen a lot of different areas of the commercial PSX codebase and have had to work with the difficulties it presented.
My main roles on this project were (and still are...):

  • Engine Design and implementation of the base technology
  • Renderer Design and implementation of a deferred rendering pipeline using Sony's GNM/GNMX APIs
  • Integration and expansion of a custom particle management system
  • Glyph based text rendering system
  • PostFx (Such as Screen Space Water...)
  • Design and creation of graphics related asset conditioning tools (joint development with other members of the team)
  • Memory management

We hope to see this project published in the future. It's great fun to work on a project that lets you work in an existing commercial codebase whilst having to develop a full suite of technology for the modern presentation of the game.

This project also won first place in the Games Technology category at the Game Republic Student Showcase 2019.
