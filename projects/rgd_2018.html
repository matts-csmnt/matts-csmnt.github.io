[C++] Retrospective Ghost Detective


Retrospective Ghost Detective was a co-op mystery solving game built using Sony’s PhyreEngine tools and C++. The game was a collaboration between 5 final year BSc programming and 4 Game design/art students.

In this project I took on the role of Graphics, UI programmer & build manager, creating split screen render passes and displaying of game elements (including a text reading system), and managing the game’s repositories and build systems.

I Worked closely with the artists to tweak the visuals of the game, and also with other programmers on the team helping them to implement their visual systems.

The game went on to win SHU’s class of 2018 best game award which was judged by local developers. They praised the game’s boldness to implement a narrative story, and UI designs that helped push this, in a short development time.
